# Capstone Project { SCISSOR }

## AltSchool Final Project

#### Scissor is a simple tool which makes URLs as short as possible.

### Getting Started

1. Prerequsites:

* Make sure you have npm installed on your local PC.

 2. Installation: 

```
npm install
```
 3. Running the Application:

```
npm run dev
```
